# Modern App Interaction Skill

## What I do
I upgrade standalone project pages into modern SaaS-grade application interfaces with:
- Command palette (Ctrl+K / Cmd+K)
- Workspace layout (sidebar + main + inspector)
- Onboarding flow
- Toast notifications
- Keyboard shortcuts
- History/recent runs
- Empty/loading/error/success states
- Export functionality

## When to use me
- When a project page looks like a simple form instead of a product
- When users need keyboard shortcuts and quick actions
- When projects need consistent interaction patterns
- When upgrading from demo to product-grade apps

## Required checks
1. Command palette opens with Ctrl+K
2. Keyboard shortcuts work
3. Toast notifications appear for actions
4. Empty states have helpful guidance
5. Loading states show progress
6. Error states have recovery suggestions
7. History persists in localStorage
8. Export produces valid Markdown/JSON

## Failure handling
- If command palette breaks, provide fallback button
- If keyboard shortcuts conflict, use alternative keys
- If localStorage fails, show warning but don't crash

## Output format
- Updated Astro/React components
- CSS for new interaction patterns
- JavaScript for keyboard handling
- Documentation for shortcuts
