# Spline 视觉方向指南

## 整体风格
暗金 + 深灰 + 3D 高级质感

## 各项目视觉方向

### 1. Portfolio Hub
主题：个人工程宇宙 / 多项目中枢
视觉元素：
- 中央发光核心
- 环绕项目轨道
- 玻璃卡片
- 金色网格
- 数据流线

### 2. LifePilot
主题：生活计划星盘 / 日历与任务流
视觉元素：
- 时间轨道
- 任务卡片
- 习惯圆环
- 温暖但高级的暗金光

### 3. InterviewPilot
主题：AI 面试舱 / 对话分析室
视觉元素：
- 双人对话面板
- 简历卡片
- 技能雷达
- 面试问题浮层

### 4. RepoPilot
主题：GitHub 仓库网络 / 安全扫描雷达
视觉元素：
- 仓库节点
- 代码分支
- 安全盾牌
- 扫描光束

### 5. KnowledgePilot
主题：知识图谱 / RAG 引用网络
视觉元素：
- 文档节点
- chunk 网络
- citation 连线
- 检索光束

### 6. OpsPilot
主题：API 运维控制台 / 服务监控矩阵
视觉元素：
- 服务器集群
- API 流线
- 状态灯
- 日志面板

### 7. Utilities
主题：开发者工具箱 / 几何组件库
视觉元素：
- 工具模块
- 格式化符号
- 色彩令牌
- 几何积木

## 性能要求
- 首页最多 1 个重型 Spline 场景
- 其它场景使用 lazy 或静态 fallback
- 移动端优先 fallback
- prefers-reduced-motion 降级
