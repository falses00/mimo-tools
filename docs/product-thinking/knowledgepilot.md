# KnowledgePilot - 苏格拉底式产品自问

## 1. KnowledgePilot 解决谁的问题？

KnowledgePilot 的目标用户是：
- **知识工作者**：需要整理大量文档和笔记
- **研究人员**：需要检索论文和资料
- **开发者**：需要查阅技术文档
- **学生**：需要整理课程资料

这些用户的共同痛点：**文档太多，找不到需要的信息**。

## 2. 用户为什么不用普通笔记 App？

普通笔记 App 的问题：
1. **搜索弱**：只能关键词搜索，不能语义搜索
2. **无引用**：搜索结果没有来源引用
3. **无问答**：不能直接问问题
4. **无 RAG**：没有检索增强生成

KnowledgePilot 解决这些问题：**文档上传 → chunk → 索引 → 搜索 → 问答 → 引用**。

## 3. 用户输入什么？

用户输入：
- **文档**：上传 Markdown/TXT/JSON 文件
- **问题**：输入自然语言问题

## 4. 系统输出什么？

系统输出：
- **搜索结果**：相关文档片段
- **问答结果**：基于文档的回答
- **引用来源**：每个回答都有引用

## 5. 它为什么不是 demo？

| 维度 | Demo | KnowledgePilot |
|------|------|----------------|
| 上传 | 无 | 支持多种格式 |
| 索引 | 无 | 自动 chunk 和索引 |
| 搜索 | 简单关键词 | 语义搜索 |
| 问答 | 无 | 基于文档的回答 |
| 引用 | 无 | 完整引用链 |
